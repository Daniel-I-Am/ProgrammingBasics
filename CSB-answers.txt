12.	Bij een synchroon HTTP request wordt de pagina nog actief geladen in de browser. Dan wordt aan de web server een request gedaan en is de render engine nog niet klaar met het schetsen van de website layout.
Een asynchroon HTTP request wordt gestart doordat de user iets doet in de DOM (zoals een knop aanklikken) of omdat een stuk code in de javascript een request aanroept. De pagina is dan al wel geladen, dus voor de gebruiker al zichtbaar. Een asynchroon request wordt op de achtergrond uitgevoerd, zonder dat de gebruiker er per se op moet wachten.
